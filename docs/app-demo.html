Coruro

A local-first desktop dashboard for your Git repositories — a Kanban board of your local repos, enriched with GitHub data and on-device AI summaries powered by Apple Intelligence. Everything runs on your machine; nothing about your code leaves it.

What it is

Coruro scans a folder of Git repositories and lays them out as cards on a five-column Kanban board (Inbox · Backlog · Active · Review · Done). Each card is an information-dashboard: language, sync state, stats, and an AI-generated summary + tags — so you can read a project at a glance without opening it.

Features

  • Editorial repo cards — language-tinted header, branch sync glance (dirty / ahead / behind / CI), and an adaptive stat grid.
  • On-device AI analysis — Apple's FoundationModels LLM generates a one-line summary and topic tags per repo, on your machine, with no network calls and no API keys.
  • GitHub enrichment — stars, forks, open issues, PRs, CI status, latest release, topics, license, cached locally and auto-refreshed.
  • Local Git status — current branch, dirty/clean, ahead/behind upstream, read-only.
  • Kanban workflow — drag repos between columns; per-repo notes timeline stored alongside the repo.
  • Quick actions — open in editor, terminal, Finder, or on GitHub.

Tech stack

Tauri 2 (Rust) · React 19 + TypeScript · Zustand · Tailwind CSS 4 · Apple FoundationModels (Swift) · vitest.

Build from source

# 1. Install JS dependencies
npm install

# 2. Build the Swift AI sidecar
./scripts/build-ai-sidecar.sh

# 3. Run in development
npm run tauri dev

# …or build a release bundle
npm run tauri build
